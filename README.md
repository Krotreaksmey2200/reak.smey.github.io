## рҹҡҖ Features

- **бһ•бҹ’бһ“бҹӮбһҖбһ–бһ·бһҮбһӮбһҺбһ·бһҸ**: бһӣбһёбһҳбһёбһҸ, бһҠбҹҒбһҡбһёбһңбҹҒ, бһўбһ¶бҹҶбһ„бһҸбҹҒбһҖбҹ’бһҡбһ¶бһӣ, бһҹбһҳбһёбһҖбһ¶бһҡбһҢбһёбһ•бҹҒбһҡбҹүбһ„бҹӢбһҹбҹ’бһҷбҹӮбһӣ
- **бһ•бҹ’бһ“бҹӮбһҖбһ’бһҡбһҺбһёбһҳбһ¶бһҸбҹ’бһҡ**: бһңбҹүбһ·бһ…бһ‘бҹҗбһҡ, бһ”бҹүбһ¶бһҡбҹүбһ¶бһ”бһјбһӣ, бһўбҹҒбһӣбһёбһ”, бһўбҹҠбһёбһ–бҹӮбһ”бһјбһӣ
- **бһңбһ·бһүбҹ’бһүбһ¶бһҹбһ¶бһҡбһҸбҹ’бһҡбҹҖбһҳбһ”бҹ’бһҡбһЎбһ„**: бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҹбһ„бҹ’бһӮбһҳ бһ“бһ·бһ„ бһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһ–бһ·бһҸ
- **бһҡбһјбһ”бһҳбһ“бҹ’бһҸбһӮбһҺбһ·бһҸбһңбһ·бһ‘бҹ’бһҷбһ¶**: бһ”бһ„бҹ’бһ бһ¶бһүбһҠбҹ„бһҷ MathJax
- **бһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһ‘бҹҶбһ“бһҫбһ”**: Responsive design бһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„ CSS Grid бһ“бһ·бһ„ Flexbox

## рҹ“ҡ Learning Topics

### бһ•бҹ’бһ“бҹӮбһҖбһ–бһ·бһҮбһӮбһҺбһ·бһҸ (Calculus)
- бһ…бҹҶбһ“бһҪбһ“бһҖбһ»бҹҶбһ•бҹ’бһӣбһ·бһ… (Real Numbers)
- бһӣбһёбһҳбһёбһҸбһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ (Limits of Functions)
- бһҠбҹҒбһҡбһёбһңбҹҒбһ“бҹғбһўбһ“бһ»бһӮбһҳбһ“бҹҚ (Derivatives)
- бһўбһ¶бҹҶбһ„бһҸбҹҒбһҖбҹ’бһҡбһ¶бһӣ (Integrals)
- бһҹбһҳбһёбһҖбһ¶бһҡбһҢбһёбһ•бҹҒбһҡбҹүбһ„бҹӢбһҹбҹ’бһҷбҹӮбһӣ (Differential Equations)

### бһ•бҹ’бһ“бҹӮбһҖбһ’бһҡбһҺбһёбһҳбһ¶бһҸбҹ’бһҡ (Geometry)
- бһңбҹүбһ·бһ…бһ‘бҹҗбһҡбһҖбҹ’бһ“бһ»бһ„бһӣбҹҶбһ  (Vectors in Space)
